About the tutorial

This is official page constituting the tutorial of the Mobile Robots (EMOR) course at The Faculty of Electronics and Information Technology, Warsaw University of Technology, realized as TRS.

TRS is an Open-source Recipe for Teaching/Learning Robotics with a Simulator (for more details please see http://ulgrobotics.github.io/trs)

For more information regarding lectures please refer to students' individual pages: http://studia.elka.pw.edu.pl.

Please visit this page frequently and check for updates!

Mettings in the semester 16L

There are six meetings in the semester:
  1. 21 March 2016 - set-up, announcement of Project 1
  2. 31 March 2016 - consultations, announcement of Project 2
  3. 4 April 2016 - consultations, Project 1 deadline, announcement of Project 3
  4. 18 April 2016 - consultations, Project 2 deadline
  5. 9 May 2016 - consultations
  6. 16 May 2016 - Project 3 deadline

Each meeting is a 4 hour workout/consultations, from 4:15 pm to 8:00 pm.

Please consider the presented projects as your homework. Implementing during the consultations hours only may result in the sortage of time.

As the time for consultations is limited, please prepare before the meeting.

There are no other consultation hours.

A deadline for each project is the last meeting when you can get positive points for the solution. Missing a deadline results in 0 points for the project. The deadlines will not be extended. All meetings end at 8:00 pm.

About the projects

Each project is a Matlab program that controls a mobile robot: KUKA youBot. The program should use the data from variuos sensors to control the velocities of the robot base wheels to realize the specified task.

The robot is controlled in a simulated environment inside the v-rep simulator. The Matlab program that controls the youBot robot communicates with the simulated hardware using v-rep Matlab bindings. The control program may use API functions from Peter Corke's Robotics, Vision and Control toolkit.

The students should run the simulator and Matlab on their own laptops, as some of the work is intended to be done individually: as homework. Both Linux and Windows operating systems are supported.

Please refer to a page that explains how to setup a laptop to work on the project.

Some informative examples of robot control programs are presented in the Example Project page.
