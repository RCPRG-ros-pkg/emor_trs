Description

The goal of the first tutorial is to program a youBot to move along egdes of a square. You can control one of the youBot that can move around a virtual scene. You need to provide the youBot with a program that allows it to move along egdes of a square and visit all vertices. When the robot reaches the first vertex, it stops.

The lenght of the edge of the square should be 1m. You have to modify file exercise01.m
