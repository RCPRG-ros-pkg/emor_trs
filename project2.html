Description

The goal of the second EMOR tutorial is to write a program enabling youBot to mova along walls, as presented in the figure below. The robot should use laser scanners in order to detect the wall and avoid collisions

Similarly to the previous tutorial you should write your program on the basis of specially prepared exercise02.m script, by modifying the section between the % EMOR exercise 01 begin and % EMOR exercise end comments.

And analogically, we also prepared a simulation environment in V-REP stored in file exercise02.ttt. You must run it before running your matlab script.
